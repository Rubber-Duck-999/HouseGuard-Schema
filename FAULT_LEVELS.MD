# Fault levels for failures

Fault levels are used to diagnose what is the seriousness of what is being raised

| Topic - Failure   | Level    |
|-------------------|----------|
| Failure Network   | 5        |
| Failure Component | 3        |
| Failure Access    | 6        |
| Motion Detected   | 7        |
| Failure Database  | 4        |
| Failure Camera    | 2        |
